My first and favorite language: Javascript. I started in web development by stringing together jQuery applications and recently learned to love Angular (and am liking React so far). Being able to write JS on the backend (Node) has also really improved my productivity. Typescript is a great addition to any JS project for more stable, less buggy code.
No one ever accused me of being an artist, but knowledge of CSS is a must in the web development world. Frameworks like Bootstrap help get you started with layout and responsive design while preprocessers like SASS are great for modularizing your CSS and avoiding one large, hard to navigate style file.
The combination of Python's simple syntax and powerful nature, along with Django's amazing web framework and ORM make this skill set very useful for getting a well organized, secure backend for a single page application up and running quickly. Looking for a more traditional request-response application? Django's templating engine has that covered too!
My new friend- the Microsoft stack. With the creation of .NET Core, MS has really moved into present day web development. It's great for creating descriptive, easy to reason about APIs. I'm currently working on one in conjunction with a latest generation Angular single page application and they work great together!
